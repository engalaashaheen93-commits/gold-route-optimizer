# ⚡ Gold Route Optimizer
### نظام دعم قرار اختيار مسارات شحن المعادن الثمينة بين آسيا ودبي

نظام ذكاء اصطناعي يقارن مسارات شحن الذهب ويوصي بالمسار الأمثل باستخدام
خوارزمية **TOPSIS** مدعومةً بنموذج لغوي كبير لتوليد توصية مُعلّلة بالعربية.

**إعداد:** م. علاء شاهين — **إشراف:** د. صارم يونس
ماجستير إدارة التقانة — الجامعة الافتراضية السورية

---

## 🚀 النشر على Streamlit Cloud (مجاناً)

### 1. أنشئ مستودعاً على GitHub
- ادخل على github.com وسجّل حساباً (إن لم يكن لديك)
- اضغط **New repository**
- الاسم: `gold-route-optimizer`
- اختر **Public**
- اضغط **Create repository**

### 2. ارفع الملفات
- اضغط **uploading an existing file**
- اسحب كل ملفات هذا المجلد ثم **Commit changes**

### 3. انشر التطبيق
- ادخل على **share.streamlit.io**
- سجّل الدخول بحساب GitHub
- اضغط **New app** → اختر المستودع
- **Main file path:** `app.py`
- اضغط **Deploy**

خلال دقيقتين سيصبح لديك رابط مثل:
`https://gold-route-optimizer.streamlit.app`

يعمل فوراً بوضع المحاكاة دون الحاجة لأي مفاتيح.

### 4. (اختياري) لتفعيل البيانات الحقيقية
في Streamlit Cloud: **Settings → Secrets** والصق:
```toml
APP_MODE = "live"
ANTHROPIC_API_KEY = "sk-ant-..."
OPENWEATHER_API_KEY = "..."
DATALASTIC_API_KEY = "..."
```

## 📁 هيكلية الملفات

| الملف | الوظيفة |
|-------|---------|
| `app.py` | الواجهة الرئيسية (Streamlit) |
| `analyzer.py` | محرك التحليل — ينسّق المزوّدين ويبني المسارات |
| `topsis.py` | خوارزمية TOPSIS |
| `providers.py` | محوّلات مصادر البيانات (mock / live) |
| `claude_advisor.py` | توليد التوصية النصية |
| `pdf_export.py` | تصدير تقرير PDF |
| `config.py` | الإعدادات والأوزان والمسارات |

## ⚙️ أوضاع التشغيل

| الوضع | الوصف |
|-------|-------|
| `mock` | بيانات محاكاة — يعمل دون أي مفاتيح (الوضع الافتراضي) |
| `hybrid` | الطقس حي + الباقي محاكاة |
| `live` | جميع البيانات حية (يتطلب مفاتيح) |

---

## 🧠 خطوات TOPSIS

1. بناء مصفوفة القرار (المسارات × المتغيرات)
2. التطبيع الإقليدي لتوحيد المقاييس
3. ضرب القيم في الأوزان
4. تحديد الحل المثالي والحل الأسوأ
5. حساب المسافة الإقليدية عن كل منهما
6. حساب درجة القرب النسبي — الأعلى هو التوصية
